== Description ==

Customer can share product review from shop page.

== Installation ==
- Download the Plugin (in a zip file) on your system
- In the Admin Panel click on Plugin and select ‘Add New’ from the sub menu
- In the next screen Upload the plugin by browsing to the location where you have downloaded the plugin
- Activate the plugin through the 'Plugins' menu in WordPress

Or 

- Upload the woo-shop-product-review-popup (directory included) to the /wp-content/plugins/ directory
- Activate the plugin through the 'Plugins' menu in WordPress


== Screenshots ==

1. Customer can share review from shop page by ADD REVIEW button.
2. After click review pop up will come to submit review.
